:line

undump command :h3

[Syntax:]

undump dump-ID :pre

dump-ID = ID of previously defined dump :ul

[Examples:]

undump mine
undump 2 :pre

[Description:]

Delete a dump that was previously defined with a "dump"_fix.html
command.  This also closes the file associated with the dump.

[Restrictions:] none

[Related commands:]

"dump"_dump.html

[Default:] none
