unfix command :h3

[Syntax:]

unfix fix-ID :pre

fix-ID = ID of a previously defined fix :ul

[Examples:]

unfix 2
unfix lower-boundary :pre

[Description:]

Delete a fix that was previously defined with a "fix"_fix.html
command.

[Restrictions:] none

[Related commands:]

"fix"_fix.html

[Default:] none
